# PDF 파싱 JSON 출력 스키마 가이드

이 문서는 "건설기술인 경력증명서 PDF → JSON" 파서의 최종 출력 데이터 구조(Schema)를 정의합니다.

## 1. 개요
본 파서는 PDF 원본의 데이터를 구조화된 JSON으로 변환합니다. 모든 날짜는 `YYYY-MM-DD` 형식을 지향하며, 추출 실패 시 빈 문자열(`""`) 또는 빈 리스트(`[]`)를 반환하여 데이터 일관성을 유지합니다.

---

## 2. 관리번호 (Primary Key)
*   **관리번호 (`관리번호`)**: 문서의 고유 식별 번호입니다. PDF 1페이지 좌측 상단에서 추출됩니다.
*   **형식**: 숫자 문자열 (예: `"123456"`)
*   **용도**: 데이터베이스 저장 시 **Primary Key**로 사용하도록 설계되었습니다.

---

## 3. 루트 객체 구조 (Root Object)

| 필드명 | 타입 | 설명 |
| :--- | :--- | :--- |
| `관리번호` | String | 문서 고유 번호 (PK) |
| `인적사항` | Object | 성명, 생년월일, 주소 |
| `서류출력일자` | String | 증명서 발급/출력일 (YYYY-MM-DD) |
| `등급` | Object | 직무/전문분야별 기술 등급 |
| `국가기술자격` | Array | 취득한 국가기술자격 목록 |
| `학력` | Array | 학위 및 졸업 정보 목록 |
| `교육훈련` | Array | 이수한 교육훈련 과정 목록 |
| `상훈` | Array | 포상 및 표창 내역 목록 |
| `벌점및제재사항` | Object | 벌점 및 행정 제재 현황 |
| `근무처` | Array | 소속 회사 및 근무 기간 목록 |
| `기술경력` | Array | 상세 설계·시공 경력 목록 |
| `건설사업관리및감리경력` | Array | 상세 CM/감리 경력 목록 |
| `공사종류별인정일수` | Array | 요약: 공사종류별 총 인정일수 |
| `직무전문분야별인정일수` | Array | 요약: 직무/전문분야별 총 인정일수 |
| `업무수행기간` | Object | 분야별 총 업무 수행 기간 요약 |
| `용역완성비율` | Object | 최근 1년 용역 완성 비율 및 참여 건수 |
| `_파싱오류` | Object | (선택적) 데이터 누락/타입 오류 정보 |
| `_검증` | Object | PDF 원본 건수와 파싱 결과 비교 검증 데이터 |
| `_검증_기술경력_페이지별` | Array | 페이지별 틸드(~) 개수 기반 파싱 정합성 검증 |

---

## 4. 상세 필드 정의

### 4.1 인적사항
*   `성명`: String
*   `생년월일`: String (YYYY-MM-DD)
*   `주소`: String

### 4.2 등급
*   `설계시공_등_직무분야`: String
*   `설계시공_등_직무분야_등급`: String
*   `설계시공_등_전문분야`: String
*   `설계시공_등_전문분야_등급`: String
*   `건설사업관리_직무분야`: String
*   `건설사업관리_직무분야_등급`: String
*   `건설사업관리_전문분야`: String
*   `건설사업관리_전문분야_등급`: String
*   `품질관리_등급`: String

### 4.3 기술경력 / 건설사업관리및감리경력
*   `사업명`: String
*   `발주자`: String
*   `공사종류`: String
*   `참여기간_시작일`: String (YYYY-MM-DD)
*   `참여기간_종료일`: String (YYYY-MM-DD 또는 "근무중")
*   `인정일수`: String (숫자)
*   `참여일수`: String (숫자)
*   `직무분야`: String
*   `전문분야`: String
*   `담당업무`: String
*   `책임정도`: String
*   `직위`: String
*   `공사(용역)개요`: String

### 4.4 요약 정보 (Summary)
*   **공사종류별인정일수**
    *   `공사종류`: String
    *   `인정일수`: String (숫자)
*   **직무전문분야별인정일수**
    *   `직무분야`: String
    *   `전문분야`: String
    *   `인정일수`: String (숫자)
*   **업무수행기간 / 용역완성비율**
    *   건설사업관리, 감리 등에 대한 요약 통계 정보 (일수, 비율 등)를 포함합니다.

### 4.5 검증 데이터 (Validation & Quality)
*   **_검증**: 각 섹션별 PDF 원본에서 카운트한 기대 건수(`PDF원본`)와 실제 추출된 건수(`파싱결과`)를 비교합니다.
*   **_검증_기술경력_페이지별**: 기술경력 파싱 시 페이지 단위로 `~`(틸드) 개수를 분석하여 누락 여부를 정밀 검증합니다.
